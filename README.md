# Fedora-Chef Dockerfile (rosstimson/fedora-chef)

This repository contains a **Dockerfile** to install [Chef][chef] using
the omnibus installer on the latest [Fedora image][fedora-image]. It
is mainly used with [Test Kitchen][test-kitchen] and saves time by
pre-installing Chef rather than having to install it at every run.
